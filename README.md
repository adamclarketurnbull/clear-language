# Clear Language Tool

A readability checker and rewriter for Layton Methodist Church volunteers.

## How to use

1. Open the tool in your browser
2. Enter your [Anthropic API key](https://console.anthropic.com/) on first use — it is saved only in your browser
3. Paste your text into the left panel
4. Select the platform you are writing for
5. Click **Check my text**
6. Copy the suggested edit from the right panel

## Reading age guide

| Badge | Reading age | What it means |
|---|---|---|
| 🟢 Green | 11 or under | Ideal — most people can read this easily |
| 🟡 Amber | 12–13 | Acceptable |
| 🔴 Red | 14+ | Too complex — simplify further |

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch**, select `main`, folder `/` (root)
4. GitHub will publish the site — the URL appears in the Pages settings

## Your API key

Your key is stored only in your browser's local storage. It is sent directly to Anthropic and nowhere else. You can remove it at any time by clicking **change** next to the key indicator.
